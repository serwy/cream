CREAM Navigator - Cognitive Reliability and Error Analysis Method
Software by Roger D. Serwy
email: serwy@uiuc.edu

Under the guidance of Dr. Esa Rantanen
email: esa.rantanen@rit.edu

CREAM is an error analysis method originally developed by Erik Hollnagel. The method is quite powerful, but cumbersome to do by hand. To help solve this problem, this software tool was developed. It is still under heavy development, but is quite usable. It requires the Firefox web browser. Using other browsers may work, but it's not likely.
